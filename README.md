# ⚛️ Quantum-Gauge-Field-Dynamics
An animated visualization of a quantum-inspired lattice gauge field using Python, NumPy, and Matplotlib. Features smooth field evolution, neon vector dynamics, and a dark scientific aesthetic.

✨ Features

⚛️ Quantum-inspired gauge field visualization
🎥 Smooth real-time animation
🖤 Dark scientific theme
💖 Neon vector field aesthetics
📊 2D lattice representation
🐍 Built entirely with Python

📸 Preview

The simulation displays a lattice of vectors where each arrow represents the local direction of a gauge field. As the animation progresses, the field evolves smoothly, creating dynamic patterns across the lattice.
🧠 How It Works

1. Create a Lattice
A two-dimensional grid is generated to represent discrete points in space.
2. Generate a Phase Field
Each lattice point is assigned a random phase value.
3. Construct the Gauge Field
The phase values are converted into complex gauge field vectors using:
Math
e^{i\phi}
4. Visualize the Field
The real and imaginary components are displayed as arrows using Matplotlib's quiver plot.
5. Animate the Dynamics
The phase field evolves over time, causing the gauge field vectors to rotate and change direction smoothly.
